Built like it's ours to keep

Because it is. No SaaS tier, no "contact sales", no telemetry phoning home.

🔐

Real RBAC, not a checkbox

Roles, permissions, per-server subusers with scoped access, and API keys that only carry the permissions you actually grant them — not the whole account.

🛡️

2FA on every login

TOTP with a real QR code, a dedicated verification screen instead of a bolted-on field, and rate-limited login attempts so brute force gets nowhere.

🖥️

A console that reconnects

Live output and power state auto-recover from dropped sockets, sleeping laptops and daemon restarts — with backoff, not a dead tab waiting for F5.

🌐

Multi-node from day one

One master panel, any number of daemon nodes talking over a token-authenticated protocol. Add capacity by pointing install.sh at a new box.

🥚

Modular Eggs

Minecraft, Discord/Telegram bots, generic Python apps, or your own Docker image — the deploy flow doesn't care what's inside the container.

🗄️

Database provisioning built in

Register a MySQL/MariaDB host once, then hand servers their own database and credentials on demand — no manual GRANT statements.

⏱️

Real cron schedules

Power actions and command chains on a schedule, claimed atomically so two ticks never double-fire the same job.

📦

Actual semantic versioning

Every release is a deliberate, tagged milestone — not "update available" on literally every commit like it used to be everywhere else.

One command. That's the whole install.

Ubuntu 22.04/24.04, root, and about two minutes.

install.sh 
bash <(curl -sSL https://raw.githubusercontent.com/superbodik/PowerNode/main/install.sh)

Already running PowerNode? Same command, one flag:

update 
PANEL_UPDATE=1 bash <(curl -sSL https://raw.githubusercontent.com/superbodik/PowerNode/main/install.sh)
